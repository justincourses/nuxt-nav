Cursor 是世界领先的 AI 代码编辑器，让开发者的生产力提升到前所未有的高度。最新版本 Cursor 2.0 引入了革命性的 Composer 界面和专用编程模型。

## 核心优势

### Agent - 将想法转化为代码
人类-AI 程序员，比任何单独开发者都更高效，提升数个量级的生产力。Agent 能够理解完整项目上下文，自主执行复杂任务，多文件编辑和创建，从想法到实现的完整流程。

### Tab - 魔法般精准的自动补全
自定义 Tab 模型，以惊人的速度和精度预测你的下一步操作。实时代码补全、预测编辑位置、上下文感知建议、多行代码生成。通过在线强化学习训练，21% 更少建议，28% 更高接受率。

### Composer - 全新对话界面
专为 Agent 工作设计的全新界面和首个编程模型。自然语言描述需求，完整项目级别理解，多文件同步编辑，实时预览更改，支持 GPT-5、Claude Sonnet 4.5 等最新模型。

### 完整代码库理解
语义搜索让 Cursor 理解你的代码库，无论规模和复杂度。深度代码库分析、智能符号引用、跨文件关联、架构模式识别、依赖关系追踪、@ 提及功能引用。

## 主要功能

**AI 编程功能**
- Cmd+K 快捷编辑 - 选中代码快速修改、内联代码生成、即时重构
- 终端 AI 助手 - 终端中使用 Cmd+I、自然语言执行命令
- 代码理解 - CodeLenses 一键理解/重构、高亮代码直接询问

**生态系统集成**
- GitHub - PR 审查和自动修复
- Slack - 团队协作助手
- BugBot - 自动发现问题、智能修复建议

**企业级功能**
- SOC 2 认证、企业级安全控制
- 团队账户管理、使用分析洞察
- 超过 50% 的 Fortune 500 公司使用

## 模型支持

支持所有主流 AI 模型：OpenAI（GPT-5.2、GPT-5、o1、o3）、Anthropic（Claude Sonnet 4.5、Opus 4.5）、Google（Gemini 3 Pro）、xAI（Grok Code）。支持 BYOM（自带模型）、API 密钥配置。
